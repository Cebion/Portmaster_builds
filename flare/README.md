# Flare Engine

## Source: 

https://github.com/flareteam/flare-engine

## Controls:

Start			= Enter
Select			= Back
A			= Attack 1
B			= Attack 2
X			= Belt 3
Y			= Belt 4
L1			= Belt 1
R1			= Belt 2
L2			= Page Forward
R2			= Page Back
L3			= Access Menu
DPAD Up			= Character
DPAD Down		= Log
DPAD Left		= Powers
DPAD Right		= Inventory
Joystick		= Move

## Building

```
git clone https://github.com/flareteam/flare-engine.git # clone the latest source code
git clone https://github.com/flareteam/flare-game.git # and game data
cd flare-engine 
mkdir build && cd build
cmake ..
make -j8
```
