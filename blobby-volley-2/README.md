# Controls



## Needed libraries

You have to supply following libraries in the libs folder
- GL4ES lib libGL.so.1
- libphysfs.so.1
 
## Building

```
git clone https://github.com/danielknobe/blobbyvolley2.git
mkdir build && cd build
cmake ..
make -j8
```
Copy folders:
- data
