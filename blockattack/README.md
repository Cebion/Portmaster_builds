# Controls



## Needed libraries
You have to supply following libraries in the libs folder
- libboost_program_options.so.1.67.0
- libphysfs.so.1
 
## Building

```
git clone https://github.com/blockattack/blockattack-game.git
./packdata.sh
cmake .
make -j4
```
Copy folders:
- data
- mods
