# Controls



## Needed libraries

You have to supply following libraries in the libs folder
- GL4ES libs
- SDL12.Compat libs
- libjpeg.so.8
- libGLEW.so.2.1
- libfluidsynth.so.2
- libinstpatch-1.0.so.2
- libjack.so.0
- libjbig.so.0
- liblzma.so.5
- libmad.so.0
- libmikmod.so.3
- libreadline.so.8
- libSDL_gfx.so.15
- libSDL_image-1.2.so.0
- libSDL_mixer-1.2.so.0
- libtiff.so.5
- libtinfo.so.6
- libwebp.so.6
- libzstd.so.1
 
## Building

```
git clone https://gitlab.com/freedroid/freedroid-src.git
./autogen.sh
./configure

```
Copy folders:
- data
- locale
- lua_modules
