## Needed libraries

You have to supply following libraries in the libs folder
- libSDL2_gfx-1.0.so.0
 
## Building

```
git clone https://github.com/pjasicek/OpenClaw.git
mkdir build && cd build 
cmake ..
make -j8
```
Copy folders:
- ASSETS

Copy files:
- SAVES.XML
- MENU.xml
- console02.tga
- config.xml
- CLAW.REZ
- clacon.ttf
- startup_commands.txt
