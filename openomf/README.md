## Needed libraries

You have to supply following libraries in the libs folder
- libxmp.so.4
- libenet.so.7
- libconfuse.so.2
- libargtable2.so.0
 
## Building

```
git clone https://github.com/omf2097/openomf.git
mkdir build && cd build
cmake ..
make -j8
```

Copy folders:
- plugins 
- gamedata
